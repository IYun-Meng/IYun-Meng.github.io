Shell脚本的编写

注: 脚本文件名建议以.sh结尾

基本结构

脚本声明:#!/bin/bash
脚本注释:#对该脚本的描述
脚本命令:命令序列

接收用户参数

在执行脚本文件后直接加上参数, 即脚本文件 参数1 参数2 参数3,...

$n第n个用户参数
$#统计用户参数个数
$*所有用户参数

判断用户参数

条件测试语句的执行格式:[ 表达式 ]
检查语句执行结果:echo $?
正确返回0, 错误返回非0.

文件测试语句

-d测试文件是否为目录类型
-e测试文件是否存在
-f判断是否为一般文件
-r测试当前用户是否有权限读取
-w测试当前用户是否有权限写入
-x测试当前用户是否有权限执行

逻辑测试语句

&&前面的命令执行成功后才会执行它后面的命令
||当前面的命令执行失败后才会执行它后面的命令
!把条件测试中的判断结果取相反值

整数值比较语句

-eq是否等于
-ne是否不等于
-gt是否大于
-lt是否小于
-le是否等于或小于
-ge是否大于或等于

字符串比较语句

=比较字符串内容是否相同
!=比较字符串内容是否不同
-z判断字符串内容是否为空

-p参数用于向用户显示一定的提示信息.
read -p "提示信息" 变量名

Shell脚本的运行

bash命令

bash是个shell脚本解释器

 bash [选项]
 |__ -n --检测脚本是否正确, 并不执行脚本.
 |__ -x --执行脚本, 输出执行过程.
 |__ -c --bash从字符串中读入命令, 如果字符串后还有变量就被设定为从$0开始的位置参数.
