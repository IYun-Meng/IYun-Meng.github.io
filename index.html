分区划分

手动划分

根分区:/
size >= 5GB
生产环境: size = 磁盘总额的4% ~ 6%

交换分区:swap
当物理内存 < 8GB时, size = 物理内存 x 1.5;
当物理内存 >= 8GB时, size = 8GB ~ 16GB;

系统引导分区:/boot
size = 200MB

应用分区:/usr
生产环境: size = 磁盘总额的10%

数据分区:/data
生产环境: size = 磁盘总额的30% ~ 40%
